# MrFrydae Tap

## How do I install these formulae?

`brew install mrfrydae/tap/<formula>`

Or `brew tap mrfrydae/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mrfrydae/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
